### 문제접근방법

부모 -> 자식 관계를 나타낼수있도록 그래프를 형성합니다.
root 노드를 입력하면 자식 노드를 탐색할 수 있는 재귀코드를 작성합니다


### 코드
```js
def dfs(cur_node):
  global cnt
  if visit[cur_node] or cur_node == delete_node:
    return

  visit[cur_node] = True

  for child in graph[cur_node]:
    if child != delete_node:
      dfs(child)
  
  if len(graph[cur_node]) == 1 and graph[cur_node][0] == delete_node:
    cnt+=1

  if len(graph[cur_node]) == 0:
      cnt+=1  

n = int(input()) 
cnt = 0 #리프노드
arr = list(map(int,input().split()))
delete_node = int(input())
graph = [[] for _ in range(n)]
visit = [0]* n
root_node = 0

for i in range(len(arr)):
  if arr[i] != -1:
    graph[arr[i]].append(i)
  else:
    root_node = i

dfs(root_node)
print(cnt)


```
